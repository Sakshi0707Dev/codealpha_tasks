# Iris Flower Classification

CodeAlpha Data Science Internship — Task 1.

This project builds a Random Forest classifier to predict iris flower species using sepal and petal measurements.

## Dataset
The dataset is loaded using Scikit-learn’s built-in Iris dataset, which is equivalent to the downloadable version provided in the task instructions.

## Model
RandomForestClassifier

## Metrics
Accuracy Score, Precision, Recall, F1-score.

## How to Run
pip install -r requirements.txt  
python iris.py
